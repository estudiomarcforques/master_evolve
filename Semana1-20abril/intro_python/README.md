# Intro to Python

## Description
This repository contains introductory materials for Python programming as part of the EVOLVE Data Science course (Week 1, April 20). It includes notebooks for learning and source code examples.

## Project Structure
- `notebooks/`: Jupyter notebooks with tutorials and exercises.
- `src/`: Source code files and scripts for Python examples.

## Getting Started
1. Clone the repository.
2. Install Python (version 3.x recommended).
3. Run notebooks using Jupyter: `jupyter notebook`.

## Usage
Explore the notebooks to learn Python basics. Modify and run the code in `src/` for practice.

For more details, refer to the course materials.
